# 剪映草稿匯出指南

將 ArcReel 已生成的影片片段按集匯出為剪映（JianYing）草稿檔案，在剪映桌面版中直接開啟並進行二次編輯——調整節奏、新增字幕、轉場、配音等。

## 前置條件

- 已在 ArcReel 中完成至少一集的影片片段生成
- 本地已安裝 **剪映桌面版**（5.x 或 6+）

## 操作步驟

### 1. 找到剪映草稿目錄

匯出前需要知道本地剪映草稿的存放路徑。

**macOS：**
```
/Users/<使用者名稱>/Movies/JianyingPro/User Data/Projects/com.lveditor.draft
```

**Windows：**
```
C:\Users\<使用者名稱>\AppData\Local\JianyingPro\User Data\Projects\com.lveditor.draft
```

> **提示**：可在剪映設定中檢視"草稿路徑"的位置。如果你修改過預設路徑，請使用實際的草稿目錄。

### 2. 在 ArcReel 中發起匯出

1. 開啟目標專案
2. 點選右上角 **匯出** 按鈕
3. 選擇 **匯出為剪映草稿**

### 3. 填寫匯出引數

| 引數 | 說明 |
|------|------|
| **集數** | 選擇要匯出的集（多集專案會出現下拉選擇器） |
| **剪映版本** | 選擇 **6.0+**（推薦）或 **5.x**，需與本地安裝的剪映版本匹配 |
| **草稿目錄** | 填入上面找到的剪映草稿路徑（首次填寫後會自動記憶） |

點選 **匯出草稿**，瀏覽器會下載一個 ZIP 檔案。

### 4. 解壓到草稿目錄

將下載的 ZIP 檔案解壓到上面填寫的剪映草稿目錄中。解壓後的結構如下：

```
com.lveditor.draft/
├── ... (其他已有草稿)
└── {專案名}_第{N}集/          ← 解壓出來的資料夾
    ├── draft_info.json        (剪映 6+) 或 draft_content.json (5.x)
    ├── draft_meta_info.json
    └── assets/
        ├── segment_S1.mp4
        ├── segment_S2.mp4
        └── ...
```

### 5. 在剪映中開啟

1. 開啟（或重啟）剪映桌面版
2. 在"草稿"列表中找到新出現的 **{專案名}\_第{N}集** 草稿
3. 雙擊開啟即可在時間線上看到所有影片片段

## 匯出內容說明

### 說書模式（Narration）

- **影片軌**：所有已生成的影片片段按順序排列
- **字幕軌**：自動附帶每個片段對應的小說原文作為字幕（白色文字、黑色描邊），可在剪映中自由調整樣式和位置

### 劇集模式（Drama）

- **影片軌**：按場景順序排列所有已生成的影片片段
- 不附帶字幕（多角色對話場景的字幕結構較複雜，建議在剪映中手動新增）

### 畫布尺寸

自動根據專案設定確定：
- 豎屏（9:16）→ 1080×1920
- 橫屏（16:9）→ 1920×1080

如專案未設定寬高比，會從首個影片檔案自動檢測。

## 常見問題

### 剪映中看不到匯出的草稿？

- 確認 ZIP 解壓到了正確的草稿目錄
- 確認解壓後的資料夾直接位於草稿目錄下（不要多套一層資料夾）
- 嘗試重啟剪映

### 版本不匹配怎麼辦？

匯出時選擇的剪映版本必須與本地安裝版本對應：
- 剪映 6.0 及以上 → 選擇 **6.0+**
- 剪映 5.x → 選擇 **5.x**

如果選錯了版本，重新匯出並選擇正確版本即可。

### 部分影片片段缺失？

匯出僅包含已成功生成的影片片段。如果某些片段尚未生成或生成失敗，它們不會出現在草稿中。回到 ArcReel 補充生成後重新匯出即可。
