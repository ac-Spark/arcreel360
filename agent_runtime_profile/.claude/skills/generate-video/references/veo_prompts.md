# Veo 3.1 Video Generation Prompt Guide

Best practices for creating effective prompts for Veo 3.1 video generation.

## Prompt Structure

按照 Veo 最佳實踐，prompt 應包含以下元素（自然融合，不使用標籤）：

1. **Composition** 構圖：鏡頭型別（wide shot, close-up, medium shot）
2. **Subject** 主體：場景描述，包含角色、環境、物體
3. **Action** 動作：角色在做什麼
4. **Dialogue** 對話：Speaker（manner）說道："text"
5. **Sound Effects** 音效：自然融入場景描述
6. **Camera** 鏡頭：運動方式的自然描述
7. **Ambiance** 氛圍：光線和情緒

**重要**：不要在 prompt 中寫入以下內容（這些透過 API 引數傳遞）：
- 影片時長（如 "8 秒"）
- 寬高比（如 "16:9"、"9:16"）

## Dialogue and Audio

### 對話格式
```
男人（手握獵刀）說道："這不是普通的熊。"
女人（聲音因恐懼而緊繃，四處張望）說道："那是什麼？"
```

使用引號包裹對話內容，在括號中描述動作和說話方式。

### 說話方式描述
- `輕聲`, `低語`, `大喊`, `喃喃自語`
- `溫柔地`, `緊張地`, `堅定地`
- `低沉的男聲`, `清脆的女聲`

### 音效（自然融入）
不使用 "音效：" 標籤，而是自然描述：
```
一聲粗獷的吠叫，樹枝折斷的聲音，潮溼泥土上的腳步聲。一隻孤鳥鳴叫。
```

```
輪胎尖銳的剎車聲，引擎轟鳴。
```

### 關於 BGM
- **不要在 prompt 中描述背景音樂**
- BGM 透過 `negative_prompt` 引數自動禁止
- 後期配樂使用 `/compose-video` 處理

## Camera Movement

| 英文術語 | 中文描述 |
|---------|---------|
| static | 鏡頭靜止 |
| pan left/right | 鏡頭向左/右平移 |
| tilt up/down | 鏡頭向上/下傾斜 |
| dolly in/out | 鏡頭緩緩推進/拉遠 |
| track left/right | 鏡頭向左/右跟蹤移動 |
| crane up/down | 鏡頭升起/降落 |
| handheld | 手持鏡頭輕微晃動 |

## Shot Types

| 英文術語 | 中文術語 | 適用場景 |
|---------|---------|---------|
| extreme close-up | 特寫 | 情緒、細節 |
| close-up | 近景 | 面部、對話 |
| medium shot | 中景 | 上半身、對話 |
| full shot | 全景 | 全身 |
| wide shot | 遠景 | 環境、建立鏡頭 |
| aerial | 俯瞰 | 鳥瞰視角 |

## Negative Prompts

使用 `negative_prompt` API 引數排除不想要的元素：
- ❌ 不要使用否定語言："no walls"
- ✅ 直接描述不想要的內容："walls, frames, borders"

預設 negative prompt（自動應用）：
```
background music, BGM, soundtrack, musical accompaniment
```

## Examples

### 對話與氛圍場景
```
wide shot，迷霧籠罩的太平洋西北森林。兩個疲憊的徒步者，一男一女，穿過蕨類植物叢，男人突然停下，盯著一棵樹。close-up：樹皮上有新鮮的深深的爪痕。男人（手握獵刀）說道："這不是普通的熊。" 女人（聲音因恐懼而緊繃，四處張望）說道："那是什麼？" 一聲粗獷的吠叫，樹枝折斷的聲音，潮溼泥土上的腳步聲。一隻孤鳥鳴叫。
```

### 詳細描寫場景
```
close-up 電影鏡頭跟隨一個絕望的男人，他身穿破舊的綠色風衣，在一面粗糙的磚牆上撥打轉盤電話，籠罩在綠色霓虹燈詭異的光芒中。鏡頭緩緩推進，揭示他下巴的緊繃和臉上刻滿的絕望，他掙扎著試圖打通電話。淺景深聚焦在他緊皺的眉頭和黑色轉盤電話上，背景模糊成霓虹色彩和模糊陰影的海洋，營造出緊迫感和孤立感。
```

### 動畫風格場景
```
一個歡快卡通風格的 3D 動畫場景。一隻有著雪豹般皮毛、大大的富有表情的眼睛、友好圓潤外形的可愛生物，歡快地在異想天開的冬季森林中蹦跳。場景中有圓潤的積雪覆蓋的樹木、輕柔飄落的雪花，以及透過樹枝灑下的溫暖陽光。生物彈跳的動作和燦爛的笑容傳達出純粹的喜悅。明亮、歡快的色彩和活潑的動畫，溫馨治癒的基調。
```

### 圖生影片場景
```
一段超現實、電影感的微距影片。小小的衝浪者在石頭洗手池裡乘著永不停歇的翻滾波浪。一個老式黃銅水龍頭正在流水，製造出無盡的海浪。鏡頭緩緩平移過這個異想天開、陽光明媚的場景，微型角色熟練地在碧綠的水面上划水。
```
